# Missions des Co-présidents — regroupement (à répartir)

Source : `sources/Missions US MONTEILHANDBALL.xlsx` (onglet 1 : **Co Présidents**)

| Grand item | Missions (détail) | Interfaces principales | Notes / livrables | CoPrez A | CoPrez B |
|---|---|---|---|---|---|
| Gouvernance & représentation | Assurer la direction collégiale avec l’autre co-président<br>Représenter officiellement l’association (partenaires, institutions, fédérations, collectivités)<br>Participer aux AG / CA / Bureaux<br>Signer les documents officiels (contrats, conventions, etc.) | Autre co-président<br>Bureau / CA<br>Ligue / Comité / Fédération<br>Mairie / collectivités<br>Partenaires | Point hebdo coprésidents |  |  |
| Conformité & cadre associatif (juridique/administratif) | Assurer la conformité aux règlements fédéraux et obligations légales (CMCD, affiliation, etc.)<br>Superviser la rédaction/màj des statuts, projet associatif, règlement intérieur<br>Superviser registres officiels, PV, conformité des décisions (avec secrétaire)<br>Déclarer les modifications (bureau/statuts/siège) en préfecture (avec secrétaire) | Secrétaire Général(e)<br>Bureau / CA<br>Instances (Ligue/Comité/Fédé)<br>Préfecture | Livrables : statuts, RI, projet associatif, PV/registre |  |  |
| Finances & ressources | Élaborer / suivre / valider le budget annuel (avec trésorière)<br>Garantir la bonne utilisation des ressources financières<br>Rechercher des financements (subventions, mécénat, partenariats) | Trésorier(e)<br>Secrétaire Général(e)<br>Commission Partenariats / sponsoring<br>Collectivités / financeurs | Clarifier : qui pilote subventions vs contribution/copilotage |  |  |
| RH & fonctionnement interne | Gestion de la relation avec le(s) salarié(s) (suivi, points réguliers) | Salarié(s)<br>Référent RH / gouvernance<br>Bureau | Rituels : points réguliers, suivi heures |  |  |
| Pilotage sportif & coordination des activités | “Superviser” équipes sportives, entraîneurs, bénévoles (respect du projet associatif)<br>Participer à la planification des événements sportifs (compétitions, tournois, stages…)<br>Favoriser la cohésion entre collectifs<br>Structurer la filière sportive (filière filles/garçons, segmentation, responsables) | Commission sportive / Responsable technique<br>Responsables de filière / catégories<br>Entraîneurs / bénévoles<br>Ententes | À définir : sens de “supervision” + outils de pilotage (suivi, indicateurs, rituels) |  |  |
| Communication & relations extérieures | Valoriser l’image du club (médias, réseaux, événements) en lien avec la com’<br>Développer relations avec partenaires (sponsors, médias, collectivités)<br>Relations presse/collectivités (selon organisation) | Commission Communication<br>Responsable Partenariats<br>Presse / médias<br>Collectivités | Clarifier : qui “fait” vs qui “valide” |  |  |
| Stratégie & développement (vision) | Définir le projet club à moyen/long terme<br>Anticiper besoins futurs (infrastructures, effectifs, encadrement)<br>Contribuer à la stratégie de développement (adhérents, nouvelles activités, options scolaires, ententes, services civiques…)<br>Impulser une dynamique et les valeurs du club | Bureau / CA<br>Commission sportive<br>Établissements scolaires / partenaires<br>Collectivités | Livrables : feuille de route, priorités annuelles |  |  |
